### 🏠 [프로젝트 방문](https://ilovenews.vercel.app/)

> 신속한 뉴스 검색과 편리한 스크랩 기능을 갖춘 앱으로 최신 뉴스를 손쉽게 관리하세요.

## 프로젝트 소개
Bing News Search API 를 사용한 뉴스검색, 스크랩 기능을 담고있는 프로젝트입니다.<br>
저의 전반적인 프로젝트 세팅, 코딩 스타일 등을 검토하는 용도로 봐주시면 감사하겠습니다.
